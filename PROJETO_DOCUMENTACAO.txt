📡 API ENDPOINTS (8 total)
================================================================================

UPLOAD:
  POST /api/upload
    - Recebe: arquivo (multipart/form-data)
    - Retorna: { session_id, filename, rows, columns, col_types, preview }
    - Status: ✅ Tested

DATA RETRIEVAL:
  GET /api/data/{session_id}/kpis
    - Retorna: { kpis: [{ title, total, mean, trend }] }
    - Status: ✅ Tested

  GET /api/data/{session_id}/stats
    - Retorna: pandas describe() em formato JSON
    - Status: ✅ Tested

  GET /api/data/{session_id}/quality
    - Retorna: { quality: [{ column, dtype, null_count, unique_count, ... }] }
    - Status: ✅ Tested

CHARTS:
  POST /api/charts/{session_id}/temporal
    Payload: { date_col, metric_col, granularity? }
    - Retorna: { data: [{ date, value, cumulative }] }
    - Status: ✅ Tested

  POST /api/charts/{session_id}/cross
    Payload: { cat_col, num_col, agg_fn?, top_n? }
    - Retorna: { data: [{ category, aggregated_value }] }
    - Status: ✅ Tested

  GET /api/charts/{session_id}/correlation
    - Retorna: { columns: [...], data: [[...], ...] }
    - Status: ✅ Tested

EXPORT (📦 NOVO):
  GET /api/export/{session_id}/excel
    - Retorna: arquivo Excel (.xlsx) com blob binary
    - Status: ✅ Tested

  GET /api/export/{session_id}/csv
    - Retorna: arquivo CSV (.csv) com formato UTF-8
    - Status: ✅ Tested


🎨 INTERFACE - 5 TABS
================================================================================

1. 📊 KPIs
   - Displays top 4 numeric metrics
   - Shows: total, mean, trend indicator
   - Cards with color-coded indicators
   - Grid: 1 col (mobile) → 4 cols (desktop)

2. 📈 Temporal
   - Time series visualization
   - Dual Y-axis (daily values + cumulative)
   - Bar chart + overlay line chart
   - Plotly interactive (zoom, pan, hover)

3. 🔍 Explorador
   - Cross-tabulation analysis
   - Categorical × Numeric aggregation
   - Top N categories (default: 20)
   - Aggregation functions: sum, mean, count, max, min

4. 🔗 Correlação
   - Heatmap of numeric variable correlations
   - Viridis color scale (-1 to +1)
   - Requires ≥2 numeric columns
   - Interactive hover shows exact values

5. ✓ Qualidade
   - Data quality metrics per column
   - Type detection, null analysis
   - Unique value counts
   - Sample values preview
